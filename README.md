# Phylogeny as a proxy for structural linguistic diversity: a reasonable approach?


This is the repository for the code accompanying the paper "Phylogeny as a proxy for structural linguistic diversity: a reasonable
approach?" presented at Evolang26.

## Code
To replicate the analysis, first, run the scripts in processing to fetch and process the respective data sources. In the folder processing/similarity, first run flatten_matrices.R and then get_measures.R. After this, the scripts in analysis can be run.

## Supplementary Materials
The supplementary materials accompanying the paper are available as a pdf: `Supplementary_materials.pdf`
